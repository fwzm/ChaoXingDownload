# ChaoXingDownload

超星学习通课程资源直链下载油猴脚本，基于 [ColdThunder11](https://github.com/ColdThunder11) 原版 v0.37 重构更新（v1.4.1）。

> **致谢**：本项目基于 ColdThunder11 的原始工作，感谢前辈开源精神！

## 功能特性

- **资源类型**：PPT/PPTX、DOC/DOCX、XLS/XLSX、PDF、MP4、FLV、AVI、WMV、MP3、ZIP/RAR/7Z 等
- **智能检测**：自动检测页面中的资源 ID，支持多种页面结构
- **模态框选择**：点击「下载」按钮弹出资源列表，勾选后下载
- **批量下载**：「批量下载」一键下载当前页面所有资源
- **PDF快速下载**：按键盘 D 键或点击「PDF全部」批量下载所有 PDF
- **进度显示**：使用 XHR 下载并显示进度条，无需跳转页面
- **多域名支持**：支持 chaoxing.com、edu.cn、xueyinonline.com 等域名
- **课程资料页**：在 coursedata 页面为每个资源添加下载按钮
- **新版总览页**：`/mycourse/stu` 页面顶部显示固定工具栏 + 右下角悬浮按钮
- **API 自动拦截**：自动拦截页面 fetch/XHR 请求，提取课程资料列表
- **手动 API 获取**：若自动拦截失败，可点击「API获取」手动拉取
- **备用下载**：CORS 限制时自动使用 GM_xmlhttpRequest

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)（推荐）或 Violentmonkey / Greasemonkey
2. 点击安装脚本：
   - [GitHub 源](https://github.com/fwzm/ChaoXingDownload/raw/main/cxdownload.user.js)

## 使用方法

### 课程学习/节点详情页面

- 页面加载后左侧显示三个悬浮按钮：
  - **下载** – 打开模态框，选择资源进行下载
  - **PDF全部** – 批量下载所有 PDF 文件
  - **批量下载** – 下载当前页面所有资源
- 按 **D** 键可快速下载所有 PDF

### 新版课程总览页面（/mycourse/stu）

- 页面顶部出现蓝色工具栏（固定定位，始终可见）
- 右下角有圆形悬浮按钮 📥 作为备用入口
- **🔍 扫描** – 扫描当前 DOM 中的文件名与 objectid
- **🌐 API获取** – 主动调用课程资料 API 获取完整资源列表（推荐）
- **📋 资源列表** – **最关键的功能**：打开弹窗列出所有找到的资源，勾选下载（不依赖 DOM 注入）
- **📄 仅PDF** – 下载所有 PDF 文件
- **⬇ 全部下载** – 批量下载所有资源

> **使用流程（重要）**：
> 1. 进入「资料」页面
> 2. 如果脚本显示「未找到资源」，点击 **🌐 API获取**
> 3. 等状态变成「API找到: X 个资源」后，点击 **📋 资源列表**
> 4. 在弹窗中勾选要下载的文件，点击「下载所选」

### 课程资料页面

- 每个资源项旁会添加绿色的「下载」按钮

## 常见问题

### 脚本没有反应？
1. 确保已安装并启用 Tampermonkey
2. 确保脚本已启用（Tampermonkey 图标应为彩色）
3. 确认您在支持的页面上（课程学习页面等）
4. 刷新页面重试

### 显示"未找到资源"？
新版 mooc2-ans 页面加载资料的方式不同，请按以下顺序尝试：
1. **切换标签触发 API** — 从「资料」切到「章节」再切回「资料」，脚本会拦截后台 API 请求
2. **点击 🌐 API获取** — 工具栏上的手动 API 拉取按钮
3. **进入具体章节** — 某些资源只在章节详情页才有下载入口
4. **刷新页面** — 确保脚本在页面加载前已启用

### 下载失败（CORS/权限错误）？
脚本会自动尝试使用 GM_xmlhttpRequest 作为备选方案。如果仍然失败，可能是：
1. 资源链接已过期
2. 需要重新登录学习通
3. 网络问题
