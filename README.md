# 学习通课程资源下载器

面向中国大陆用户的学习通/超星课程资料下载油猴脚本。当前版本重心是新版 `mooc2-ans` 课程资料页：自动识别资料资源，提供资源列表、筛选下载、批量下载和备用下载模式。

本项目基于 [ColdThunder11](https://github.com/ColdThunder11) 原版 v0.37 重构更新，协议继承 GPL-3.0。

## 当前版本

- 当前脚本版本：`v2.2.1`
- 脚本文件：[cxdownload.user.js](https://raw.githubusercontent.com/fwzm/ChaoXingDownload/master/cxdownload.user.js)
- 自动更新：脚本已内置 `@updateURL` 和 `@downloadURL`
- 适配域名：`chaoxing.com`、`xueyinonline.com`、`edu.cn` 及其子域名

## 主要功能

- **中文界面**：顶栏、悬浮面板、资源弹窗、状态提示、错误提示均已汉化。
- **首次说明弹窗**：安装或更新后首次进入支持页面，会自动弹出使用说明。
- **右下角说明入口**：忘记操作流程时，点击右下角「下载」按钮，再点「使用说明」。
- **资源自动识别**：从页面 DOM、同源 iframe、iframe 上报、fetch/XHR 接口返回、页面缓存中合并资源 ID。
- **重复 UI 防护**：iframe 中只扫描并上报资源，不再创建第二个顶栏或第二个悬浮按钮。
- **资源列表缓存**：首次打开时获取资源信息；当前页面未刷新或关闭时，再次打开不会重复请求。
- **资源列表弹窗**：每个文件都可独立勾选，并提供「全选」「反选」「清空」后下载所选文件。
- **类型筛选**：顶部支持 `PPT`、`PDF`、`DOC` 快速筛选。
- **批量下载**：点击「全部下载」触发当前已识别资源的批量下载。
- **浏览器 / IDM 下载**：默认直接交给浏览器下载；安装 IDM 扩展后可由 IDM 自动接管。
- **自动回退**：保留油猴下载模式，但遇到 `not_whitelisted`、超时或其他错误时会自动切换到浏览器 / IDM。
- **文件名处理**：优先保留学习通原始文件名；无法识别时使用 `资源_objectid` 作为备用名称。
- **自动更新地址**：用户无需手动填写 Tampermonkey 的更新 URL。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)。
2. 打开脚本安装地址：
   - [GitHub Raw 安装](https://raw.githubusercontent.com/fwzm/ChaoXingDownload/master/cxdownload.user.js)
3. Tampermonkey 弹出安装页后，点击「安装」。
4. 后续更新可在 Tampermonkey 中点击「检查用户脚本的更新」。

## 使用流程

1. 打开学习通课程，进入「资料」页。
2. 等顶部蓝色工具栏显示资源数量，例如「3 个资源」。
3. 点击顶部「资源列表」。
4. 在弹窗中勾选要下载的文件。
5. 点击「下载所选」。

资源信息只会在当前页面首次打开列表时获取。关闭弹窗后再次点击「资源列表」，会直接使用本页内存缓存；刷新或关闭网页后缓存自动清空。

如果顶部显示「未找到资源」：

1. 先点击顶部「扫描」。
2. 仍未找到时，点击右下角橙色「下载」按钮。
3. 在悬浮面板中点击「缓存扫描」。
4. 资源数量出现后，再点击「资源列表」下载。

## 页面按钮说明

### 顶部蓝色工具栏

- **扫描**：重新扫描当前页面和已缓存资源。
- **资源列表**：打开资源选择弹窗，可逐项勾选，也可全选、反选或清空后下载。
- **PPT / PDF / DOC**：只显示对应类型文件。
- **全部下载**：批量下载当前已识别的全部资源。
- **模式：浏览器/IDM / 模式：油猴**：切换下载方式；推荐保持浏览器 / IDM 模式。

### 右下角橙色「下载」按钮

点击后打开备用面板：

- **扫描**：同顶部扫描。
- **资源列表**：同顶部资源列表。
- **PDF**：只显示 PDF 文件。
- **全部下载**：同顶部全部下载。
- **缓存扫描**：从接口缓存、内联数据、iframe 上报中补充资源。
- **使用说明**：重新打开使用说明。
- **模式：浏览器/IDM / 模式：油猴**：同顶部模式切换，并与顶部按钮同步。

### 资源旁边的「下载」按钮

在部分课程资料列表或章节资料中，脚本会在资源行旁边插入小的「下载」按钮，用于单个资源下载。

## 下载模式

- **浏览器 / IDM**：默认模式，直接打开资源下载地址。Chrome 可使用自带下载器；安装并启用 IDM Integration Module 后，IDM 可接管下载。
- **油猴下载**：可选模式，使用 Tampermonkey 的 `GM_download`。若遇到域名白名单限制、超时或调用错误，脚本会自动回退到浏览器 / IDM。

浏览器 / IDM 模式不再通过 `GM_xmlhttpRequest` 把文件转成 Blob，因此不受油猴下载域名白名单影响，也便于 IDM 扩展识别原始下载地址。

## 常见问题

### Tampermonkey 右上角显示数字 2，是不是装了两个脚本？

通常不是。学习通页面里有 iframe，同一个脚本可能在主页面和 iframe 中各运行一次，所以 Tampermonkey 图标会显示 `2`。当前版本中 iframe 只负责上报资源，不会创建第二个顶栏或第二个悬浮按钮。

### 刷新时会不会闪出两个顶栏？

v2.2.1 会在 `document-start` 尽早注入旧版 UI 隐藏规则，并清理旧版顶栏/悬浮按钮。正常情况下最终只保留一个顶部工具栏和一个右下角「下载」按钮。

### 点击下载时出现 Tampermonkey 跨域确认页怎么办？

v2.1.8 起已经补充明确的 `@connect` 域名，普通学习通资源一般不再需要用户手动确认。若资源被学习通跳转到新的未知域名，Tampermonkey 仍可能弹出确认页，这是扩展的安全机制。

### 显示「未找到资源」怎么办？

按这个顺序处理：

1. 确认当前在课程「资料」页。
2. 点击顶部「扫描」。
3. 点击右下角「下载」→「缓存扫描」。
4. 从「资料」切到其他标签页再切回来，触发学习通重新请求接口。
5. 刷新页面后重试。

### 下载失败怎么办？

可能原因：

- 学习通登录状态过期，需要重新登录。
- 资源链接临时失效，需要刷新页面重新获取。
- 浏览器阻止了多个文件自动下载，需要在地址栏提示中允许该网站下载多个文件。
- IDM 未接管时，请确认已安装并启用 `IDM Integration Module`；否则会由浏览器自带下载器处理。
- 学校或网络环境限制了资源访问。

## 面向维护者的当前实现要点

- `@run-at document-start`：尽早注入，减少旧 UI 闪烁。
- `iframe worker`：iframe 只执行扫描和 `postMessage` 上报，不创建 UI。
- 资源来源合并：DOM、同源 iframe、跨 iframe 上报、fetch/XHR 捕获、页面缓存。
- 单例 UI ID：当前版本使用 `__cxdl_bar_unique_v221` 和 `__cxdl_float_unique_v221`。
- 下载模式偏好：保存在 `localStorage.cxdl_dlMode`。
- 首次说明弹窗标记：保存在 `localStorage.cxdl_help_seen_v221`。
